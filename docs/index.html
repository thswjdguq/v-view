왜 만들었나?

"혼자 하는 면접 연습엔
피드백이 없다"

❌ 기존 방법

  • 면접 코치 — 비싸다 (건당 5~10만원)
  • 친구 도움 — 불편하고 주관적
  • 혼자 녹화 — 뭘 개선해야 할지 모름

✅ v-view

  • AI가 직종·자기소개서 기반 질문 생성
  • 실시간 시선 분석으로 집중도 측정
  • 세션 종료 후 즉시 리포트 + 개선 포인트
20초. 문제 → 기존방법 한계 → v-view 대안 흐름으로.
